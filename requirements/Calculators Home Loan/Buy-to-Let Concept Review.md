# Buy-to-Let - Concept Review Pack

This package contains three design directions for the Buy-to-Let rebuild using Home Loan design standards as the baseline.

## Scope locked in all concepts
- Single-pane flow (calculation + explanation merged).
- Existing Buy-to-Let calculation structure retained (price, rent, deposit, mortgage type, advanced assumptions).
- Dedicated "Calculation Inputs" section removed.
- Key Results table includes an "Include Advanced Options" control to reveal/hide advanced result rows.
- Mobile-first responsive behavior with stacked layout under tablet breakpoints.

## Direction summary
- Concept A - Meridian Pulse
  - Closest to Home Loan's energetic dark visual language.
  - High-contrast result card, strong slider styling, neon-accent controls.
- Concept B - Ledger Dock
  - Clean editorial layout with conservative palette and table readability focus.
  - Most neutral option for broad audience comfort.
- Concept C - Atlas Horizon
  - Soft gradient and glass-like surfaces with modern rounded treatment.
  - Between A and B in visual intensity.
