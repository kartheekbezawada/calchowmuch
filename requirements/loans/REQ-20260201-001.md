# Requirement: Home Loan Calculator Website Testing

## REQ-20260201-001

### Objective
Ensure the Home Loan Calculator website functions correctly, meets domain rules, and provides accurate, accessible, and user-friendly calculations for end users.

### Scope
- Functional testing of all calculator features
- UI/UX and accessibility validation
- Domain rule and calculation accuracy
- Cross-browser and device compatibility
- Error handling and edge cases

### Acceptance Criteria
1. **Functional Coverage**
   - All input fields accept valid values and reject invalid ones (UI-2.3, UI-2.4, UI-2.5)
   - Calculation triggers on button click and displays results in the correct format (UI-2.6, UI-2.7)
   - Reset/clear functionality works as expected
2. **Calculation Accuracy**
   - Results match formulas and edge cases in requirements/loans/HOME_LOAN_RULES.md
   - Handles zero, negative, and large values per domain rules
3. **UI/UX & Accessibility**
   - Layout follows three-column structure (UI-1.1, UI-1.2)
   - No dropdowns; button groups for toggles (UI-2.2)
   - Inputs capped to 12 characters (UI-2.5)
   - aria-live regions for result updates (UI-3.1)
   - Keyboard navigation and focus order correct
   - Always-visible scrollbars (UI-1.3)
4. **Cross-Browser/Device**
   - Works in latest Chrome, Firefox, Safari, Edge
   - Responsive on desktop and mobile
5. **Error Handling**
   - Invalid input shows clear error messages (UI-2.4)
   - No calculation on incomplete/invalid input
6. **Compliance**
   - Passes all universal requirements in requirements/universal/UNIVERSAL_REQUIREMENTS.md
   - Calculator appears in sitemap.xml
