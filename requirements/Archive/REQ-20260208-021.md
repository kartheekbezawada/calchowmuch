# REQ-20260208-021 — Power Nap Calculator (Time & Date)

## Status: COMPLETE

- Type: Calculator
- Change Type: New Calculator + SEO
- Priority: HIGH
- SEO Impact: YES
- New route (must be added): `/time-and-date/power-nap-calculator/`

## 1) Objective

Create a standalone Power Nap Calculator page optimized for SERP intent, while preserving the calculator shell layout and universal UI rules.

Note (repo truth): Similar “power nap” functionality currently exists as a mode inside `/time-and-date/nap-time-calculator/`. This REQ is to publish a dedicated standalone page; it must not remove the existing mode (AP-2.3).

## 2) Scope

In-scope:
- New calculator page under the Time & Date section.
- Calculation logic for multiple nap duration options.
- Explanation Pane per UI-EXP-PANE-STD.
- Full SEO package per SEO_RULES.
- Navigation + sitemap coverage for a LIVE calculator (DOC-SITEMAP-1..5).

Out of scope:
- Shell layout changes (UI-3.1, UI-3.2).

## 3) Calculation Rules

Input:
- Nap start time (time of day; date rollover must be handled).

Outputs:
- Provide multiple nap options with wake-up times computed as:
  - Wake = Start + Duration

Required duration set (minimum):
- 10 minutes (micro nap)
- 20 minutes (power nap)
- 30 minutes (power nap)
- 60 minutes (recovery)
- 90 minutes (full cycle)

UX rules:
- Highlight 20–30 minutes as the best “power nap” range.
- If the start time is late evening (e.g., after 18:00 local time), show a non-medical warning that long naps may affect night sleep.

## 4) Calculation Pane Interaction Contract

- Use an explicit Calculate button.
- After initial baseline, input edits must not update results until Calculate is clicked (UI-2.6, ISS-UI-FDP-008).
- No dropdowns for duration selection; use button groups/segmented controls if needed (UI-2.5).
