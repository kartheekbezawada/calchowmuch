## 3) Calculation Rules

Constants:
- Sleep cycle length = 90 minutes
- Fall-asleep buffer = 15 minutes

Modes:

### Mode A — “I need to wake up at”

Input:
- Target wake time $W$

Compute recommended bedtimes for cycle counts $c \in \{4, 5, 6\}$:
- Bedtime $B_c = W - 90c\text{ minutes} - 15\text{ minutes}$

### Mode B — “I’m going to sleep at”

Input:
- Sleep time $S$ (the time user intends to fall asleep)

Compute recommended wake times for cycle counts $c \in \{4, 5, 6\}$:
- Wake $W_c = S + 90c\text{ minutes}$
