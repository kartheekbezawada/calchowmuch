How Much Did the Value Grow?

The short answer: percentage increase shows how far a new value moved above or below the original value.

Use this calculator when the original value is the baseline and you want to express the change as a signed percentage.

How to Guide

  1. Enter the original value in X and the new value in Y.
  2. Click Calculate percentage increase to see the signed result and direction.
  3. Check the increase amount to understand the raw movement before it is converted to a percent.
  4. Use the sign and direction together to tell whether the change is growth, decline, or no change.

Worked Example

With X = 80.00 and Y = 100.00, the increase amount is 20.00. Dividing 20.00 by 80.00 gives 0.25, and multiplying by 100 gives +25.00%.

How to Interpret Your Result

A positive result means the new value is above the original value, a negative result means it is below the original value, and a zero result means there was no change. Use the percent to compare growth across different baselines and the increase amount when you need the raw unit change.

Formula Breakdown

The percentage increase formula is:

% Increase = ((Y - X) / X) x 100

  1. Difference: Y - X = 20.00.
  2. Relative ratio: (20.00) / (80.00) gives the growth ratio over the original value.
  3. Percentage value: Multiply by 100 to get +25.00%.
  4. Direction: Use the sign to label this as Increase.

If X is 0, percentage increase is undefined because division by zero is not possible.

Frequently Asked Questions

01

What is percentage increase?

Percentage increase is how much a value has grown compared to its original value, expressed as a percentage.

02

How do you calculate percentage increase from X to Y?

Subtract X from Y, divide by X, then multiply by 100.

03

What is the percentage increase formula?

The formula is ((Y - X) / X) x 100.

04

What is the difference between increase amount and percentage increase?

Increase amount is Y - X, while percentage increase compares that change to the original value.

05

Can percentage increase be more than 100%?

Yes, if the increase is larger than the original value, the percentage increase exceeds 100%.

06

What happens if the original value is zero?

Percentage increase is undefined because you cannot divide by zero.

07

What if the new value is smaller than the original value?

Then the result is not an increase; it will produce a negative percentage, which indicates a decrease.

08

Can I use decimals for the original and new values?

Yes, the calculation works the same for decimal values.

09

Is percentage increase the same as percent change?

Not always; percent change can be increase or decrease, while percentage increase focuses on growth.

10

Where is percentage increase used in real life?

It is used for price rises, salary increases, growth rates, and comparing performance over time.
