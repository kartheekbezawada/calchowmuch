Reverse percentage helps you find the original value when you know a final value and the percentage it represents.

This reverse percentage calculator answers "Y is X% of what number?" by dividing Y by X expressed as a fraction of 100.

It is commonly used to find the original price before discount, the original total from a percentage-based result, or the base value behind a percentage.

Explanation

To solve a reverse percentage, convert the percentage into a decimal by dividing by 100, then divide the final value by that decimal.

For example, if 60 is 20% of a number, then the original number is 60 ÷ 0.20 = 300.

This is also how you find an original price before a discount when the discounted price is known as a percentage of the original.

If the percentage is zero, the original value cannot be calculated because division by zero is not possible.

Frequently Asked Questions

Q: What is reverse percentage?

A: Reverse percentage finds the original value when a final value is a known percentage of it.

Q: How do you calculate "Y is X% of what number"?

A: Divide Y by (X ÷ 100).

Q: What is the reverse percentage formula?

A: The formula is Original = (Y × 100) ÷ X.

Q: How do I find the original price before discount?

A: If the discounted price is Y and it equals X% of the original, the original is (Y × 100) ÷ X.

Q: What happens if the percentage is 0%?

A: The original value is undefined because division by zero is not possible.

Q: Can I use decimal percentages like 12.5%?

A: Yes, decimal percentages work the same way.

Q: Can the final value be negative?

A: Yes, negative values are supported mathematically.

Q: How do I check if the result is correct?

A: Multiply the original value by X% and confirm it equals Y.

Q: Is reverse percentage the same as percent change?

A: No, reverse percentage finds a base value, while percent change compares two values.

Q: Where is reverse percentage used in real life?

A: It is used for discounts, taxes, price recovery, and finding totals from percentage-based values.
