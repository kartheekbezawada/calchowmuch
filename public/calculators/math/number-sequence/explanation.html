What is a Number Sequence?

A number sequence is an ordered list of numbers following a specific pattern. This calculator supports two common types: arithmetic and geometric sequences.

When to Use This Calculator

  • Finding the nth term of a predictable pattern
  • Calculating compound growth scenarios (geometric)
  • Solving math problems involving sequences and series
  • Predicting future values based on a constant rate of change

Input Definitions

  • Sequence Type: Choose Arithmetic (constant difference) or Geometric (constant ratio).
  • First Term (a1): The starting value of the sequence.
  • Common Difference (d): For arithmetic sequences, the constant added to each term.
  • Common Ratio (r): For geometric sequences, the constant multiplied to each term.
  • Term Index (n): Which term to calculate (must be at least 1).
  • Preview Length (k): How many terms to display (1-50).

Formulas

Arithmetic Sequence:

an = a1 + (n - 1) * d

Geometric Sequence:

an = a1 * r(n-1)

Worked Example (Arithmetic)

Using the defaults: a1 = 2, d = 3, n = 10

  1. Formula: an = a1 + (n - 1) * d
  2. Substitute: a10 = 2 + (10 - 1) * 3
  3. Calculate: a10 = 2 + 9 * 3 = 2 + 27 = 29

First 10 terms: 2, 5, 8, 11, 14, 17, 20, 23, 26, 29

Worked Example (Geometric)

Using: a1 = 2, r = 2, n = 10

  1. Formula: an = a1 * r(n-1)
  2. Substitute: a10 = 2 * 29
  3. Calculate: a10 = 2 * 512 = 1024

First 10 terms: 2, 4, 8, 16, 32, 64, 128, 256, 512, 1024

Special Cases

  • Negative d or r: Both negative differences and ratios are allowed, creating alternating or decreasing sequences.
  • r = 1: A geometric sequence with ratio 1 produces constant terms.
  • r = 0: All terms after the first will be 0.
  • Large n with r > 1: Results grow very quickly and may exceed JavaScript number limits.

Edge Cases and Validation

  • n must be a positive integer (at least 1).
  • Preview length k is clamped to a maximum of 50.
  • Very large results will show an error message.
  • Results are formatted to 4 decimal places.
