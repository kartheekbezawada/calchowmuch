Formula Notes

The solver uses the quadratic formula x = (-b +/- sqrt(b^2 - 4ac)) / 2a. The discriminant Δ = b^2 - 4ac tells you how many roots exist and whether those roots are real or complex.

The value of a must not be zero. If a = 0, the equation is no longer quadratic and this solver stops with an error.

Worked Example

Two real roots

Use x^2 - 5x + 6 = 0, so a = 1, b = -5, and c = 6.

The discriminant is 1, so the roots are real and distinct: x = 3 and x = 2.

Repeated root

Use x^2 - 2x + 1 = 0.

The discriminant is 0, so the equation has one repeated real root: x = 1.

FAQ

What happens if a is zero?

The equation becomes linear, so this quadratic solver returns an error.

Why do I sometimes get two answers?

Most quadratics intersect the x-axis at two points, giving two roots.

Can this calculator handle decimal inputs?

Yes, all three coefficients accept decimal values.

How is the discriminant used?

The discriminant determines whether roots are real, repeated, or complex.

Does it show solving steps?

Yes, it shows formula substitution and discriminant evaluation in order.

What does a negative discriminant mean?

It means there are no real roots, only complex conjugate roots.

Are repeated roots shown clearly?

Yes, when Δ = 0, the result card reports one repeated real solution.

Can I use this for homework checking?

Yes, it is useful for verifying manual work and final roots.

Why is the method fixed to quadratic formula?

The solver uses a consistent method that works for all valid quadratics.

Do large coefficients affect output precision?

Results are formatted to readable precision while preserving key accuracy.
