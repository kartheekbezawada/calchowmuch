What Is a System of Linear Equations?

A system of linear equations is a set of two or more equations that share variables. Solving the system means finding values that satisfy every equation at the same time.

Solution Types

  • Unique solution: Lines (or planes) intersect at one point.
  • No solution: Lines are parallel or planes do not intersect.
  • Infinite solutions: The equations represent the same line or plane.

Methods Used

  • Elimination: Add or subtract equations to eliminate a variable.
  • Substitution: Solve one equation for a variable and substitute.
  • Matrix / Cramer's Rule: Use determinants to solve for variables.

2×2 System Example

For the system:

x + 2y = 5

3x + 4y = 11

The determinant is 1×4 - 2×3 = -2. Using Cramer's rule gives x = 1 and y = 2.

3×3 System Notes

For 3×3 systems, Cramer's rule uses determinants of 3×3 matrices. If the main determinant is zero, there is no unique solution.

Graphing Insight

For 2×2 systems, each equation is a line. The intersection point on the graph is the solution.
