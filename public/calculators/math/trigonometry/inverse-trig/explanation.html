Inverse Trigonometric Functions

Inverse trig functions undo sine, cosine, and tangent. They answer the question: "Which angle gives this value?" Each inverse has a restricted principal range.

Principal Ranges

  • arcsin(x) returns angles from -90 to 90 degrees.
  • arccos(x) returns angles from 0 to 180 degrees.
  • arctan(x) returns angles from -90 to 90 degrees (excluding the endpoints).

All Solutions

Because trig functions are periodic, there are infinitely many solutions. The calculator lists all solutions within a custom interval so you can match real-world constraints.

Domain Restrictions

arcsin and arccos are only defined for inputs between -1 and 1. If the value lies outside that range, no real angle exists.

Applications

Inverse trig is used to find angles from measured ratios in surveying, robotics, and signal processing. Interval selection helps align solutions with physical limits.
