What Is an Integral?

Integration is the reverse process of differentiation. An integral represents the accumulation of quantities, such as the area under a curve, total distance traveled, or accumulated change.

There are two types of integrals: indefinite integrals (antiderivatives) and definite integrals (numerical values).

Indefinite Integrals (Antiderivatives)

An indefinite integral ∫f(x)dx represents the family of all functions whose derivative is f(x). The result includes an arbitrary constant C because differentiation of a constant is zero.

Notation: ∫f(x)dx = F(x) + C, where F'(x) = f(x)

Definite Integrals

A definite integral ∫[a to b]f(x)dx calculates the net signed area between the curve f(x) and the x-axis from x=a to x=b. It produces a numerical value rather than a function.

Fundamental Theorem of Calculus: ∫[a to b]f(x)dx = F(b) - F(a)

Integration Rules

Power Rule for Integration

∫x^n dx = x^(n+1)/(n+1) + C, where n ≠ -1

Example: ∫x³ dx = x⁴/4 + C

Constant Rule

∫k dx = kx + C (constant k)

Constant Multiple Rule

∫k·f(x)dx = k·∫f(x)dx

Sum/Difference Rule

∫[f(x) ± g(x)]dx = ∫f(x)dx ± ∫g(x)dx

Common Integrals

  • ∫x^n dx = x^(n+1)/(n+1) + C
  • ∫1/x dx = ln|x| + C
  • ∫e^x dx = e^x + C
  • ∫sin x dx = -cos x + C
  • ∫cos x dx = sin x + C
  • ∫sec² x dx = tan x + C
  • ∫1/(1+x²) dx = arctan x + C

Integration Techniques

Substitution Method

Used for composite functions. Replace a portion of the integrand with u to simplify.

Example: ∫2x·e^(x²)dx, let u = x², du = 2x dx

Integration by Parts

Based on the product rule: ∫u dv = uv - ∫v du

Example: ∫x·e^x dx

Partial Fractions

Decompose rational functions into simpler fractions.

Example: ∫1/(x²-1) dx = ∫1/[2(x-1)] dx - ∫1/[2(x+1)] dx

Geometric Interpretation

For a definite integral ∫[a to b]f(x)dx:

  • The integral represents the signed area between f(x) and the x-axis
  • Area above the x-axis is positive
  • Area below the x-axis is negative
  • The net area is the algebraic sum

Worked Example

Calculate ∫(3x² + 2x - 5)dx

  1. Apply power rule to each term:
    • ∫3x² dx = 3·x³/3 = x³
    • ∫2x dx = 2·x²/2 = x²
    • ∫-5 dx = -5x
  2. Combine results: ∫(3x² + 2x - 5)dx = x³ + x² - 5x + C

Real-World Applications

  • Physics: Finding position from velocity, work from force
  • Economics: Total cost from marginal cost, consumer surplus
  • Probability: Computing probabilities from probability density functions
  • Engineering: Calculating volumes, centroids, moments of inertia

Common Mistakes

  • Forgetting the constant of integration + C for indefinite integrals
  • Incorrect application of power rule when n = -1
  • Sign errors when evaluating definite integrals
  • Not simplifying the antiderivative before evaluating bounds
