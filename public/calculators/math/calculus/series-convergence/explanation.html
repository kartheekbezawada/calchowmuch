What Is Series Convergence?

An infinite series is the sum of an infinite sequence of terms: Σan = a1 + a2 + a3 + ... The series converges if the sequence of partial sums approaches a finite limit. Otherwise, it diverges.

Understanding convergence is crucial in calculus, analysis, and applications ranging from physics to computer science.

Convergence Tests

Ratio Test

For a series Σan, compute L = limn→∞ |an+1/an|:

  • If L < 1: The series converges absolutely
  • If L > 1: The series diverges
  • If L = 1: The test is inconclusive

Best for: Series with factorials, exponentials, or powers of n

Example: For Σn!/nn, the ratio test quickly determines convergence

Root Test (Cauchy Root Test)

For a series Σan, compute L = limn→∞ n√|an|:

  • If L < 1: The series converges absolutely
  • If L > 1: The series diverges
  • If L = 1: The test is inconclusive

Best for: Series with nth powers

Example: For Σ(n/(n+1))n, the root test is ideal

Comparison Test

Compare your series Σan with a known series Σbn:

  • Direct Comparison: If 0 ≤ an ≤ bn and Σbn converges, then Σan converges
  • Direct Comparison: If an ≥ bn ≥ 0 and Σbn diverges, then Σan diverges
  • Limit Comparison: If limn→∞ (an/bn) = L where 0 < L < ∞, then both series converge or both diverge

Common comparison series:

  • p-series: Σ1/np converges if p > 1, diverges if p ≤ 1
  • Geometric series: Σrn converges if |r| < 1, diverges otherwise

Common Series

Convergent Series

  • p-series (p > 1): Σ1/n2, Σ1/n3, etc.
  • Geometric (|r| < 1): Σ(1/2)n, Σ(2/3)n
  • Alternating series: Σ(-1)n/n (conditionally convergent)
  • Exponential: Σn/2n, Σn!/nn

Divergent Series

  • Harmonic series: Σ1/n
  • p-series (p ≤ 1): Σ1/√n
  • Geometric (|r| ≥ 1): Σ2n, Σ(-1)n

Step-by-Step Analysis

  1. Write out the general term an of the series
  2. Choose an appropriate convergence test based on the form of an
  3. Compute the limit or comparison required by the test
  4. Apply the test criteria to determine convergence or divergence
  5. If inconclusive, try a different test

Worked Example

Determine if Σn/2n converges using the ratio test

  1. General term: an = n/2n
  2. Next term: an+1 = (n+1)/2n+1
  3. Compute ratio: |an+1/an| = [(n+1)/2n+1] / [n/2n] = (n+1)/(2n)
  4. Take limit: L = limn→∞ (n+1)/(2n) = 1/2
  5. Since L = 1/2 < 1, the series converges by the ratio test

Common Mistakes to Avoid

  • Confusing sequence convergence with series convergence
  • Misapplying tests (e.g., using ratio test on p-series)
  • Forgetting absolute value in ratio and root tests
  • Concluding divergence when a test is inconclusive (L = 1)
  • Not simplifying limits before evaluating

Real-World Applications

  • Physics: Fourier series for wave analysis
  • Engineering: Power series approximations
  • Computer Science: Analysis of algorithms (asymptotic series)
  • Probability: Expected values and moment-generating functions
  • Finance: Present value of perpetuities and annuities
