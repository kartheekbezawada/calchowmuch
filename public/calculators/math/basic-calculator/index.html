Quickly add, subtract, multiply, or divide two numbers.

Explanation

The basic calculator combines two numbers using the four core arithmetic operations. Pick an operation and the result updates immediately.

  • Add: Combine two values (12 + 8 = 20).
  • Subtract: Find the difference (12 − 8 = 4).
  • Multiply: Scale one value by another (12 × 8 = 96).
  • Divide: Split evenly (12 ÷ 8 = 1.5).

If you enter 0 as the second number for division, the calculator will warn you because dividing by zero is undefined.
