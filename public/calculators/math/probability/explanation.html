How to Guide

  1. Select the event rule that matches the question before entering numbers.
  2. Use outcome counts for single events and percentage probabilities for combined-event modes.
  3. If events can overlap, use the inclusive OR option and enter P(A and B).
  4. For binomial mode, enter the number of trials, target successes, and success probability.

Formula Guide

The route supports P(E) = favorable / total, P(A and B) = P(A) × P(B) for independent events, P(A or B) = P(A) + P(B) - P(A and B), P(A|B) = P(A and B) / P(B), Bayes' P(A|B) = P(B|A) × P(A) / P(B), and the binomial expression P(X = k) = C(n, k) × p^k × (1-p)^(n-k).

FAQ

When should I use AND instead of OR?

Use AND when both events must happen together. Use OR when at least one of the events can happen.

Why does OR subtract overlap?

Without subtracting the shared part, the overlap would be counted twice and the result would be too large.

What is the difference between conditional mode and Bayes mode?

Conditional mode starts with the overlap directly, while Bayes mode solves the updated probability from prior and evidence values.

Can probabilities be entered above 100%?

No. Every probability input is limited to the 0% to 100% range.

What does binomial mode calculate?

It calculates the probability of exactly k successes in n independent trials with the same success probability.

Does the route show decimals and percentages?

Yes. The answer card keeps the decimal probability and the percentage view together for easy interpretation.
