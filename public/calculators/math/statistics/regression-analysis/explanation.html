Regression Analysis

Regression analysis is a statistical method used to model the relationship between a dependent variable (Y) and one or more independent variables (X).

Types of Regression

Linear Regression

Fits the equation: y = ax + b

The least squares method minimizes the sum of squared residuals to find the best-fitting line.

Polynomial Regression

Fits higher-degree polynomials: y = a0 + a1x + a2x2 + ... + anxn

Useful when the relationship between variables is curvilinear.

Exponential Regression

Fits the equation: y = a x ebx

Appropriate for data showing exponential growth or decay patterns.

Logarithmic Regression

Fits the equation: y = a + b x ln(x)

Suitable for data where growth rate decreases as X increases.

Key Statistics

R-squared (R2)

The coefficient of determination measures how well the regression line fits the data. Values range from 0 to 1, where 1 indicates a perfect fit.

  • R2 >= 0.9: Excellent fit
  • R2 >= 0.7: Good fit
  • R2 >= 0.5: Moderate fit
  • R2 < 0.5: Poor fit

Adjusted R-squared

A modified version of R2 that adjusts for the number of predictors. It only increases if a new predictor improves the model more than expected by chance.

Standard Error

Measures the average distance between observed values and the regression line. Smaller values indicate better fit.

Residuals

The differences between observed and predicted values. Analyzing residuals helps identify patterns the model may have missed.

Coefficient Testing

The t-statistic and p-value for each coefficient test whether it significantly differs from zero. A small p-value (typically < 0.05) indicates the coefficient is statistically significant.
