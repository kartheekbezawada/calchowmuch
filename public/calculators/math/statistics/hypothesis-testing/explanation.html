Hypothesis Testing Practical Guide

Use this route when you need a clean test decision with the p-value, test statistic, and supporting sample summary in one place. It helps you move from raw numbers to a structured reject-versus-fail-to-reject call without bouncing between separate t-test and chi-square pages.

  • Best for: one-sample, two-sample, paired, and chi-square significance checks.
  • Quick answer flow: choose the test, paste the data, set alpha, and read the answer card first.
  • Output includes: test statistic, p-value, degrees of freedom, decision, interval, and effect size where applicable.

How to Guide

  1. Select the test family that matches your data structure.
  2. Paste the required sample or frequency values and choose the alternative hypothesis when relevant.
  3. Choose alpha based on how strict the decision rule should be.
  4. Click Run test and read the decision line together with the p-value and interval context.

Formula Guide

The t-family compares a mean difference with the noise around that difference, while the chi-square test compares observed and expected frequencies using sum((O - E)^2 / E). In plain terms, the route asks whether the observed gap is large enough relative to normal variation to justify rejecting the null idea.

Why keep the confidence interval in view?

The interval gives scale to the decision, which helps you judge not only significance but also the plausible size of the effect.

FAQ

What does reject H0 actually mean?

It means the observed data would be unlikely enough under the null assumption that you choose the alternative explanation instead.

What if the result is not significant?

It means the data did not supply strong enough evidence against the null at the chosen alpha, not that the null is proven true.

When should I use a paired test?

Use a paired test when each value in one sample is naturally matched to a value in the other sample, such as before-and-after measurements.

Why do one-sided and two-sided tests matter?

They change how the evidence is allocated across the tails, so the p-value and decision can differ even with the same sample data.

What does Cohen's d add to the result?

It gives a scale-free effect size so you can judge whether the difference is tiny, moderate, or large in practical terms.

Can a statistically significant result still be unimportant?

Yes. Large samples can make small effects significant, so you should read the effect size and interval as well as the p-value.
