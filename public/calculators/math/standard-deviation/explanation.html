What is Standard Deviation?

Standard deviation measures the amount of variation or dispersion in a dataset. A low standard deviation indicates values are close to the mean, while a high standard deviation indicates values are spread out over a wider range.

When to Use This Calculator

  • Analyzing the spread of test scores, measurements, or survey responses
  • Comparing variability between two datasets
  • Quality control to identify outliers
  • Financial analysis to measure investment volatility

Input Definitions

  • Dataset: A list of numbers separated by commas, spaces, or newlines. Negative numbers and decimals are allowed.
  • Mode: Choose between Sample (n-1) for a sample from a larger population, or Population (n) when your data represents the entire population.

Formulas

Mean:

Mean = Sum of all values / Count of values

Variance:

Sample Variance = Sum of (each value - mean)² / (n - 1)

Population Variance = Sum of (each value - mean)² / n

Standard Deviation:

Standard Deviation = Square root of Variance

Worked Example

Using the default dataset: 10, 12, 23, 23, 16, 23, 21, 16

  1. Count: n = 8
  2. Sum: 10 + 12 + 23 + 23 + 16 + 23 + 21 + 16 = 144
  3. Mean: 144 / 8 = 18
  4. Squared differences: (10-18)² + (12-18)² + ... = 64 + 36 + 25 + 25 + 4 + 25 + 9 + 4 = 192
  5. Sample Variance: 192 / (8-1) = 192 / 7 = 27.4286
  6. Sample Standard Deviation: sqrt(27.4286) = 5.2372

Sample vs. Population

  • Sample (n-1): Use this when your data is a subset of a larger population. The (n-1) divisor (Bessel's correction) provides an unbiased estimate.
  • Population (n): Use this when your data represents the entire population you're studying.

Edge Cases and Validation

  • Sample mode requires at least 2 values (division by n-1 = 0 is undefined for n = 1).
  • Population mode works with a single value (standard deviation will be 0).
  • Non-numeric values are rejected and listed as errors.
  • Results are rounded to 4 decimal places.
