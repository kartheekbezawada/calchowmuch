How Percent Change Works

Percentage change measures how much a value has increased or decreased relative to its starting point. It's commonly used for price changes, growth rates, and performance comparisons.

Steps:

  1. Subtract the starting value from the ending value to get the difference.
  2. Divide the difference by the starting value.
  3. Multiply by 100 to convert to a percentage.

Formula:

((Ending - Starting) / Starting) x 100

Example:

If a stock price rises from $50 to $60:
Change = (60 - 50) / 50 x 100 = 20% increase

How Percentage Of Works

This calculator finds what a given percentage of a number equals. Useful for calculating tips, discounts, taxes, and portions.

Steps:

  1. Convert the percentage to a decimal by dividing by 100.
  2. Multiply the decimal by the value.

Formula:

(Percentage / 100) x Value

Example:

What is 25% of 200?
25 / 100 x 200 = 50

How X Is What % Of Y Works

This calculator finds what percentage one number (the part) represents of another number (the whole). Useful for test scores, completion rates, and proportional analysis.

Steps:

  1. Divide the part (X) by the whole (Y).
  2. Multiply by 100 to convert to a percentage.

Formula:

(X / Y) x 100

Example:

What percent is 30 of 150?
30 / 150 x 100 = 20%

How Add/Subtract Percentage Works

This calculator adds or subtracts a percentage from a value. Commonly used for applying discounts, markups, tips, and tax calculations.

Steps (Adding):

  1. Calculate the percentage amount: Value x (Percentage / 100).
  2. Add the amount to the original value.

Steps (Subtracting):

  1. Calculate the percentage amount: Value x (Percentage / 100).
  2. Subtract the amount from the original value.

Formulas:

Add: Value + (Value x Percentage / 100)
Subtract: Value - (Value x Percentage / 100)

Example:

Add 15% to $100:
100 + (100 x 0.15) = $115
