Permutation & Combination Calculator

Calculate permutations (nPr) and combinations (nCr).

Explanation

What Are Permutations and Combinations?

Both permutations and combinations count ways to select items from a set. The key difference is whether the order of selection matters.

  • Permutation: Order matters (ABC is different from BAC)
  • Combination: Order doesn't matter (ABC is the same as BAC)

When to Use Each

  • Permutations: Ranking, arranging, passwords, seating arrangements, race placements
  • Combinations: Selecting committees, lottery numbers, choosing items from a menu

Input Definitions

  • n: The total number of items in the set (must be 0-170).
  • r: The number of items to select or arrange (must be 0 to n).
  • Factorial mode: Uses only n to compute n!.

Formulas

Permutation (nPr):

nPr = n! / (n-r)!

Combination (nCr):

nCr = n! / (r! Ã— (n-r)!)

Factorial (n!):

n! = n Ã— (n-1) Ã— (n-2) Ã— ... Ã— 2 Ã— 1

Worked Example (Permutation)

Using defaults: n = 10, r = 3

  1. Formula: 10P3 = 10! / (10-3)! = 10! / 7!
  2. Simplify: 10 Ã— 9 Ã— 8 = 720
  3. Result: There are 720 ways to arrange 3 items from 10.

Example: Selecting 1st, 2nd, and 3rd place from 10 contestants.

Worked Example (Combination)

Using: n = 10, r = 3

  1. Formula: 10C3 = 10! / (3! Ã— 7!)
  2. Simplify: (10 Ã— 9 Ã— 8) / (3 Ã— 2 Ã— 1) = 720 / 6 = 120
  3. Result: There are 120 ways to choose 3 items from 10.

Example: Selecting a committee of 3 people from 10 candidates.

Key Relationships

  • nPr is always greater than or equal to nCr (since order creates more arrangements)
  • nPr = nCr Ã— r! (permutation counts each combination r! times)
  • nC0 = nCn = 1 (one way to choose nothing or everything)
  • nCr = nC(n-r) (symmetric property)

Special Cases

  • 0! = 1: By definition, zero factorial equals one.
  • nP0 = 1: One way to arrange zero items (do nothing).
  • nPn = n!: Arranging all items.
  • nC1 = n: n ways to choose 1 item.

Edge Cases and Validation

  • n must be between 0 and 170 (larger values cause overflow).
  • r must be between 0 and n.
  • Results are displayed as exact integers (no rounding).
  • Large results are shown without commas.
