Use this calculator to measure how tightly or loosely values sit around the mean. It keeps the standard deviation, variance, and mean summary in one answer-first result so you can judge spread quickly before moving on to z-scores or interval work.

Quick Answer Table

Quick answer table for the standard deviation calculator
Measure What it tells you Quick interpretation
Standard deviation The typical distance from the mean A larger value means the dataset is more spread out.
Variance The squared spread around the mean Useful in formulas, but less intuitive than standard deviation.
Sample mode Uses n - 1 in the denominator Best when the data is only a sample.
Population mode Uses n in the denominator Best when the data already covers the whole group.

Formula Guide

Standard deviation is the square root of variance. Population variance divides the squared deviations by n, while sample variance divides by n - 1 to correct for estimating spread from a sample.

FAQ

What does a larger standard deviation mean?

It means the values are spread further from the mean on average.

When should I use sample mode?

Use sample mode when the dataset is only one sample from a bigger population.

When should I use population mode?

Use population mode when the dataset already covers the whole population you care about.

Can standard deviation be zero?

Yes. If every value is the same, there is no spread, so the standard deviation is zero.

Does standard deviation show skew?

No. It shows spread, not shape, so a skewed dataset can still share the same standard deviation as another list.

Can I use decimals and negative values?

Yes. The route accepts decimals and negative numbers as long as the input is numeric.
