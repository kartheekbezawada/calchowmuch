Hypothesis Testing

Perform hypothesis tests to determine statistical significance.

Explanation

Hypothesis Testing

Hypothesis testing is a statistical method for making decisions based on data. It helps determine whether observed results are statistically significant or could have occurred by chance.

The Hypothesis Testing Framework

  1. State the hypotheses: Define null (H0) and alternative (H1) hypotheses
  2. Choose significance level (alpha): Typically 0.05, 0.01, or 0.10
  3. Calculate test statistic: Based on sample data
  4. Find p-value: Probability of observing results as extreme as the sample
  5. Make decision: Reject or fail to reject H0

Test Types

One-Sample t-Test

Tests whether a sample mean differs from a hypothesized population mean.

t = (x-bar - mu0) / (s / sqrt(n))

Use when population standard deviation is unknown.

Two-Sample t-Test (Welch's)

Compares means of two independent groups without assuming equal variances.

Welch's t-test uses adjusted degrees of freedom for unequal variances.

Paired t-Test

Tests whether the mean difference between paired observations is zero.

Used for before/after studies or matched pairs designs.

Chi-Square Goodness of Fit

Tests whether observed frequencies match expected frequencies.

chi2 = sum((O - E)2 / E)

Alternative Hypotheses

  • Two-sided: H1: mu != mu0 (mean is different)
  • Greater: H1: mu > mu0 (mean is greater)
  • Less: H1: mu < mu0 (mean is less)

Effect Size: Cohen's d

Measures the magnitude of the difference in standard deviation units.

  • |d| >= 0.8: Large effect
  • |d| >= 0.5: Medium effect
  • |d| >= 0.2: Small effect

Confidence Intervals

A 95% confidence interval means we are 95% confident the true parameter falls within the interval. If the interval excludes the null hypothesis value, the result is significant at alpha = 0.05.

Interpretation

  • p < alpha: Reject H0 - Evidence supports alternative hypothesis
  • p >= alpha: Fail to reject H0 - Insufficient evidence to reject null
