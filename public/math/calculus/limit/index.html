Limit Calculator

Calculate limits including indeterminate forms

Explanation

What Is a Limit?

A limit describes the value that a function approaches as the input approaches some value. Limits are fundamental to calculus and are used to define derivatives, integrals, and continuity.

Notation: lim[x→a] f(x) = L means "the limit of f(x) as x approaches a is L"

Types of Limits

Two-Sided Limits

The function approaches the same value from both left and right: lim[x→a] f(x) exists if lim[x→a⁻] f(x) = lim[x→a⁺] f(x)

One-Sided Limits

Left-hand limit: lim[x→a⁻] f(x) (approaching from values less than a)

Right-hand limit: lim[x→a⁺] f(x) (approaching from values greater than a)

Limits at Infinity

Describes function behavior as x gets arbitrarily large: lim[x→∞] f(x) or lim[x→−∞] f(x)

Indeterminate Forms

Situations where direct substitution yields an undefined expression:

  • 0/0: Most common; often requires algebraic manipulation or L'Hôpital's rule
  • ∞/∞: Can use L'Hôpital's rule
  • 0 × ∞: Rewrite as 0/0 or ∞/∞
  • ∞ − ∞: Combine fractions or rationalize
  • 0⁰, 1^∞, ∞⁰: Use logarithms

L'Hôpital's Rule

If lim[x→a] f(x)/g(x) yields 0/0 or ∞/∞, then:

lim[x→a] f(x)/g(x) = lim[x→a] f'(x)/g'(x)

Conditions:

  • The limit must be in indeterminate form 0/0 or ∞/∞
  • Both f'(x) and g'(x) must exist near a
  • Can be applied repeatedly if the form persists

Algebraic Techniques

Factoring

Cancel common factors to resolve 0/0 forms

Example: lim[x→2] (x²−4)/(x−2) = lim[x→2] (x+2)(x−2)/(x−2) = lim[x→2] (x+2) = 4

Rationalization

Multiply by conjugate to eliminate radicals

Example: lim[x→0] (√(x+1)−1)/x

Common Denominators

Combine fractions before taking the limit

Dominant Terms

For limits at infinity, factor out highest power

Example: lim[x→∞] (3x²+2x)/(x²−5) = lim[x→∞] (3+2/x)/(1−5/x²) = 3

Limit Laws

If lim[x→a] f(x) = L and lim[x→a] g(x) = M, then:

  • Sum: lim[x→a] [f(x) + g(x)] = L + M
  • Difference: lim[x→a] [f(x) − g(x)] = L − M
  • Product: lim[x→a] [f(x) · g(x)] = L · M
  • Quotient: lim[x→a] [f(x) / g(x)] = L / M (if M ≠ 0)
  • Power: lim[x→a] [f(x)]^n = L^n
  • Constant: lim[x→a] k = k

Continuity and Limits

A function f is continuous at x = a if:

  1. f(a) is defined
  2. lim[x→a] f(x) exists
  3. lim[x→a] f(x) = f(a)

If any condition fails, the function has a discontinuity at x = a.

Worked Examples

Example 1: Direct Substitution

Find lim[x→3] (x² + 2x − 1)

  1. Function is continuous at x = 3
  2. Substitute directly: (3)² + 2(3) − 1 = 9 + 6 − 1 = 14
  3. Answer: 14

Example 2: Factoring

Find lim[x→2] (x² − 4)/(x − 2)

  1. Direct substitution gives 0/0 (indeterminate)
  2. Factor numerator: (x + 2)(x − 2)/(x − 2)
  3. Cancel common factor: x + 2
  4. Substitute: 2 + 2 = 4
  5. Answer: 4

Example 3: L'Hôpital's Rule

Find lim[x→0] (sin x)/x

  1. Direct substitution gives 0/0
  2. Apply L'Hôpital's: lim[x→0] (cos x)/1
  3. Substitute: cos(0) = 1
  4. Answer: 1

Common Limits to Know

  • lim[x→0] (sin x)/x = 1
  • lim[x→0] (1 − cos x)/x = 0
  • lim[x→∞] (1 + 1/x)^x = e
  • lim[x→0] (e^x − 1)/x = 1
  • lim[x→0] (ln(1 + x))/x = 1

Common Mistakes

  • Applying L'Hôpital's rule when the form is not indeterminate
  • Confusing lim[x→a] f(x) with f(a) when function is discontinuous
  • Not checking if one-sided limits agree before claiming two-sided limit exists
  • Incorrectly manipulating infinity in algebraic expressions
