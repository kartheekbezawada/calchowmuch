Formula Guide

A limit asks what value a function approaches as the input approaches a target point. When direct substitution works, the route reports that value immediately. When direct substitution fails, the route compares nearby values numerically to estimate the left-hand and right-hand behavior.

For limits at infinity, the route evaluates the function at increasingly large magnitudes to check whether the outputs appear to converge, diverge, or grow without bound.

Worked Example

Direct substitution example

Enter x^2 + 2*x - 1 and let x approach 3.

The route returns 14 because the function is continuous there, so direct substitution works immediately.

Indeterminate example

Enter (x^2 - 1)/(x - 1) and let x approach 1.

Direct substitution fails, but the nearby values settle near 2, so the route reports a two-sided limit of about 2.

FAQ

What is a limit?

It is the value a function approaches as the input approaches a chosen point.

What is a two-sided limit?

It checks whether the left-hand and right-hand values approach the same result.

What is a one-sided limit?

It checks the behavior from only one side of the approach point.

Why can a two-sided limit fail?

The most common reason is that the left-hand and right-hand values approach different results.

What does direct substitution mean?

It means you can plug in the approach value directly and get a valid finite result.

What is an indeterminate form?

It is a substitution result such as 0/0 that needs more analysis before the limit can be judged.

Can I use infinity here?

Yes. Enter inf or -inf to estimate a limit at infinity.

Does the variable field matter?

Yes. The route evaluates the expression using the variable name you enter.

Does this route prove limits symbolically?

No. It focuses on practical direct substitution and numeric approximation rather than formal proof steps.

When is this route useful?

It is useful for continuity checks, approach-value comparisons, and quick calculus practice.
