Explanation

What Is a Derivative?

The derivative of a function measures the rate at which the function's value changes with respect to changes in its input. Geometrically, the derivative at a point is the slope of the tangent line to the function's graph at that point.

The derivative of f(x) with respect to x is denoted as f'(x), df/dx, or dy/dx.

Differentiation Rules

Power Rule

For f(x) = x^n: f'(x) = n¬∑x^(n-1)

Example: If f(x) = x¬≥, then f'(x) = 3x¬≤

Constant Rule

If f(x) = c (constant): f'(x) = 0

Constant Multiple Rule

If f(x) = c¬∑g(x): f'(x) = c¬∑g'(x)

Sum/Difference Rule

If f(x) = g(x) ¬± h(x): f'(x) = g'(x) ¬± h'(x)

Product Rule

If f(x) = g(x)¬∑h(x): f'(x) = g'(x)¬∑h(x) + g(x)¬∑h'(x)

Mnemonic: (first)' ¬∑ second + first ¬∑ (second)'

Common Derivatives

  • d/dx(sin x) = cos x
  • d/dx(cos x) = -sin x

  • d/dx(ln x) = 1/x

Higher-Order Derivatives

The second derivative f''(x) is the derivative of the first derivative. It measures the rate of change of the rate of change, indicating how the slope is changing. This is useful for determining concavity and inflection points.

Step-by-Step Differentiation

  1. Identify the form of the function (power, product, quotient, composition)
  2. Select the appropriate differentiation rule

Real-World Applications

  • Physics: Velocity is the derivative of position; acceleration is the derivative of velocity
  • Economics: Marginal cost/revenue represents the derivative of total cost/revenue
  • Optimization: Finding maxima and minima by setting f'(x) = 0
  • Engineering: Analyzing rates of change in systems and signals

Common Mistakes to Avoid

  • Forgetting to apply the chain rule for composite functions
  • Incorrectly applying the product rule (must use both terms)
  • Sign errors in the quotient rule
  • Treating constants as variables or vice versa
