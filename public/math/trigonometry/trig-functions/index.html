Quick Answer Table

Function What it uses Common issue to watch

sec(x), csc(x), cot(x) Reciprocals of cos, sin, and tan-related ratios. Undefined anywhere the matching base value is zero.

Formula Guide

The route evaluates the six standard trig relationships from one angle. The core identity is sin^2(x) + cos^2(x) = 1, and tangent follows from tan(x) = sin(x) / cos(x).

The reciprocal functions use the same base angle: sec(x) = 1 / cos(x), csc(x) = 1 / sin(x), and cot(x) = cos(x) / sin(x). Undefined outputs usually point to a zero denominator rather than a broken route.

FAQ

What functions does this route calculate?

It calculates sin, cos, tan, sec, csc, and cot for one angle.

Can I enter radians instead of degrees?

Yes. Switch the angle unit control to radians and enter the angle there.

Does the route show exact values?

Yes. Special angles such as 30°, 45°, and 60° show exact labels alongside decimal values.

Why can tangent be undefined?

Tangent is sine divided by cosine, so it becomes undefined when cosine equals zero.

What does amplitude change?

Amplitude stretches the graph vertically, which matters most for sine and cosine style waves.

What does period change?

Period changes how wide one cycle appears on the graph.

Why do reciprocal functions have gaps?

They are undefined where the matching sine or cosine value is zero, so the graph breaks there.

Can I use this for unit-circle review?

Yes. The angle pair, quadrant, and special-angle note make it useful for unit-circle study too.

Does the graph use the same unit as the angle input?

The entered angle can use degrees or radians, but the graph period field is displayed on a radian scale.

Is this route good for homework checking?

Yes. It is designed for quick verification, not formal symbolic proof work.
