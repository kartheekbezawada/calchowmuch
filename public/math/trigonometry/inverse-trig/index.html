Formula Guide

Inverse trig functions answer the question, which angle produces this ratio? The principal value is the one angle inside the standard inverse-trig range.

Extra solutions come from periodic trig behavior. For example, if sin(theta) = 0.5, the interval can include multiple angles such as 30° and 150°, then repeating versions separated by full rotations.

Worked Example

arcsin(0.5)

Use the default interval from -360° to 360°.

The principal value is 30°, and the interval list also includes angles such as 150°, -210°, and -330°.

arctan(1)

Set the function to arctan and the value to 1.

The principal value is 45°, then the interval list repeats by 180° because tangent has a shorter period than sine or cosine.

FAQ

What does inverse trig do?

It turns a known trig value back into the angle that produces it.

Why is there a principal value?

Each inverse function needs one standard range so the returned angle is unique.

Why can there be many solutions?

Sine, cosine, and tangent repeat, so the same trig value can match multiple angles.

What is the arcsin domain?

For real angles, the input must be between -1 and 1.

What is the arccos domain?

For real angles, the input must also be between -1 and 1.

What is the arctan domain?

Arctan accepts any real number as input.

Can I use radians?

Yes. Switch the output unit to radians and the interval fields follow that unit too.

Why does tangent repeat faster?

Tangent repeats every 180°, while sine and cosine repeat every 360°.

What happens if the interval is narrow?

The route only lists the solutions that fit inside the interval you set.

Is this route useful for surveying or robotics?

Yes. It helps convert measured ratios into angles while respecting practical angle limits.
