Z-Score Calculator

Calculate the z-score for a value given the mean and standard deviation.

Explanation

What is a Z-score?

A z-score (also called a standard score) measures how many standard deviations a value is from the mean. It standardizes values from different distributions, making them comparable.

When to Use This Calculator

  • Comparing values from different datasets
  • Determining how unusual a value is
  • Standardizing test scores
  • Quality control (identifying outliers)
  • Statistical hypothesis testing

Input Definitions

  • Value (x): The data point you want to standardize.
  • Mean (Î¼): The average of the population or sample.
  • Standard Deviation (Ïƒ): The measure of spread in the data. Must be greater than 0.

Formula

z = (x - Î¼) / Ïƒ

Where:

  • x = the value
  • Î¼ = the mean
  • Ïƒ = the standard deviation

Worked Example

Using the defaults: x = 85, Î¼ = 75, Ïƒ = 10

  1. Formula: z = (x - Î¼) / Ïƒ
  2. Substitute: z = (85 - 75) / 10
  3. Calculate: z = 10 / 10 = 1
  4. Interpretation: 85 is 1 standard deviation above the mean.

Interpreting Z-scores

  • z = 0: The value equals the mean.
  • z > 0: The value is above the mean.
  • z < 0: The value is below the mean.
  • |z| > 2: The value is considered unusual (outside ~95% of data).
  • |z| > 3: The value is considered very unusual (outside ~99.7% of data).

The 68-95-99.7 Rule

For normally distributed data:

  • ~68% of values have z-scores between -1 and +1
  • ~95% of values have z-scores between -2 and +2
  • ~99.7% of values have z-scores between -3 and +3

Common Applications

  • Test Scores: SAT, IQ, and other standardized tests use z-scores for comparison.
  • Finance: Measuring how unusual a stock return is compared to historical performance.
  • Quality Control: Identifying products that deviate too far from specifications.

Edge Cases and Validation

  • Standard deviation must be greater than 0 (division by zero is undefined).
  • All inputs must be valid finite numbers.
  • Results are formatted to 4 decimal places.
  • Negative z-scores indicate values below the mean.
