# Markup Calculator

This calculator moves between cost, selling price, and markup percentage for a single item or a basket of items. Use it when you need a fast selling price, want to audit current pricing, or need a weighted markup view across multiple products.

## How to Guide

1. Choose single-product or basket mode.
2. Choose cost-to-price or price-to-markup mode.
3. Enter the values for the active setup.
4. Review selling price, markup amount, and basket markup if multiple rows are used.

Markup is measured against cost, so it answers a different question from margin. If cost is $50 and markup is 40%, the markup amount is $20 and the selling price is $70. If you work backward from price, the calculator shows what percentage of cost that markup represents.

Basket mode is useful when individual rows vary a lot in cost. The combined result is weighted by total cost, not averaged row by row, so high-cost items influence the basket markup more than low-cost accessories.

Use full landed cost whenever possible. Freight, packaging, processing fees, and routine discounts can materially change real markup, especially on lower-priced items. The calculator gives clean pricing math, but market conditions and channel fees still decide whether a target markup is workable.

## Frequently Asked Questions

### How do you calculate price from cost and markup?

Multiply cost by one plus the markup rate as a decimal.

### How do you calculate markup from cost and price?

Subtract cost from price, divide by cost, then multiply by 100.

### Is markup the same as margin?

No. Markup is based on cost; margin is based on selling price.

### Why is basket markup not a simple average?

Because the combined result is weighted by total cost and quantity.

### Can markup be negative?

Yes. That means the selling price is below cost.

### What is the most common mistake?

Using incomplete cost inputs and overstating the true markup.
